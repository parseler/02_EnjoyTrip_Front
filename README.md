## 목차

- [프로젝트 소개](#프로젝트-소개)
- [기능](#기능)

## 프로젝트 소개

전국 관광지를 지도에서 확인하고 검색할 수 있습니다.

관광지를 핫플레이스로 등록하고 공유 게시판에서 확인할 수 있습니다.

## 기능

1. 회원가입 및 로그인 기능
2. 지역별, 관광지 유형별 관광지 검색 기능
3. 핫플레이스 등록 및 공유 게시판 업로드 기능
4. 공지 게시글 작성 기능
